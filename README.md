# CareerPilot AI

## Overview

CareerPilot AI is a multi-agent career guidance system built using Google Gemini.

## Features

* Resume Analysis
* Interview Preparation
* Skill Gap Analysis
* Learning Roadmaps

## Architecture

CareerAgent

* ResumeAgent
* InterviewAgent
* SkillGapAgent
* RoadmapAgent
* SecurityAgent
* MCP Server

## Technologies

* Python
* Google Gemini
* Google Colab

## Future Enhancements

* Streamlit Deployment
* PDF Resume Upload
* Job Matching
